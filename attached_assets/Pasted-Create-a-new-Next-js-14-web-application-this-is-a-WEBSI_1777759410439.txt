This site honours classical South Indian dance (Bharatanatyam), Carnatic music, and Kerala temple aesthetics. The design must feel rooted in tradition, dignified, and warm — never generic, never tech-trendy.

In app/globals.css:
- Set body background to the parchment color (#F4EBD9), text to ink (#1F1612)
- Add a fixed pseudo-element body::before with subtle radial gradient overlays of gold (6% opacity) and maroon (4% opacity) at opposite corners — this gives a soft "aged stone" texture
- NEVER use pure white (#FFFFFF) or pure black (#000000) anywhere — substitute parchment for white and ink for black
- Default heading style: Cormorant Garamond, weight 400 for h1/h2, weight 500 for h3
- Default body: Manrope, weight 400, line-height 1.6
